# Advanced Image Resizer 🎨📐

A high-fidelity, fully client-side image processing utility designed for designers, photographers, and print professionals. Scale assets seamlessly across digital interfaces (pixels) and physical media (inches/centimeters) at production-grade print densities (DPI).

## ✨ Key Features
* **Zero Server Uploads:** All image scaling occurs directly in your browser using high-performance HTML5 canvas resampling. Your data never leaves your device.
* **Universal Measurement Engine:** Seamlessly toggle between Pixels (`px`), Inches (`in`), and Centimeters (`cm`).
* **Print-Ready Resolution Control:** Custom DPI/PPI rendering pipeline for crisp physical outputs (300 DPI, 600 DPI, etc.).
* **State Persistence:** Built-in `SettingsManager` automatically saves your chosen preferences (measurement units, custom dimensions, DPI, and visual dark/light mode) to local storage.
* **Batch Processing:** Drop multiple images at once, process them in parallel, and export them in a single, organized `.zip` file instantly.
* **iOS-Inspired Glassmorphic UI:** A gorgeous, responsive interface featuring interactive backdrop-blur layering and native-style dark/light toggles.
